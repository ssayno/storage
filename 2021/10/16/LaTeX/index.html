Preface

All of my blogs are prepared for myself when I need reinstall system.
In this blog, I will write down the process of how to use LaTeX in Emacs.(For me, it is a diffcult process. Since I am new to Elisp, but this is very interesting to learning this, as same as LaTeX)

Usage

If you want, you can clone my configuration from here. And frist you should install the plugin use-package, and then clone this in your ~/.emacs.d, then you can write LaTeX in your emacs.

## use-package ## `use-package` is a tool that Convenient for you to use(manage) your package, just use it yestoday, may have some erro, I don't know if it can use in other places. ### auctex ### All of my configuration about auctex from [auctex](https://www.gnu.org/software/auctex/manual/auctex.pdf), after I config the basic configuration, I will upload my configuration to my [github](https://github.com/ssayno/semesterfive/tree/main/works/12).(You can put this in the same directory as `init.el`, and add `(require 'mylatex)`, I am expend it now, so you may can't use it. You should know how to change it. **Anyway, he can run on my computer:-)**)

cdlatex

cdlatex offer many shortcuts to insert LaTeX command, you can use C-c ? to know them, and you can access its github page for more information.

latex-preview-pane

This plugin is used to get the concept of what you see is what you get, maybe this concept is rejected by someone, but it is useful for mathmatical formula sometimes.(I can config it to --synctex=1)

reftex

Used for cross-referencing.(But now I don’t need this, so I did’t configure this). Reftex have been bundled with Emacs, so you should install by youself.

pdf-tools

When you use preview in tex file, Emacs will use docview to open the pdf file, but it is fuzzy, according to the guideline from Google, customize the doc-view-resolution, it will render this page lager. So you should install this plugin to make you pdf file clear. I install this plugin from this github page.

neotree

Use the sidebar from this github page.

exec-path-from-shell

If you use command like \usepackage, it will give you this erro

you should install this plugin from this [github](https://github.com/purcell/exec-path-from-shell#exec-path-from-shell) page. (I forgot `Magit` these days. I will solve it for him tomorrow)
