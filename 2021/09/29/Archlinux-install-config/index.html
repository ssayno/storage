Install ArchLinux on your computer

The system about my installation is Win10.
Most informations about the installation from this web-page.

Preparation

  • First, you should prepare a USB which memory should be greater than 8GB.
  • And you should have a correct network environment.(Maybe the compaus network is’t suitable, which you need login in a validation page, but the on the installation of ArchLinux there are no graphical interface.)
  • Install the ISO file from the ArchLinux-Downloads and then burn an installation boot disk by using Power-ISO recommended, but what I use it myself is Ultra-ISO(This is the software I used when I installed Ubuntu). On Windows, you maybe use the root to open the Ultra-ISO when you burn the instllation boot disk.
  • Access the MAINBOARD BIOS. The model of my computer is Lenovo, then you can click F12(You can get your own computer information from BaiDu), and in the Security set the Secure Boot to Disable, in the boot set the Boot Mode to UEFI rather than legacy(There were a time after I install the ArchLinux, this maybe have some mistakes), in boot, drag the USB to the first.

The formal installation

  • Check your wireless connection is disabled, in general, it is blocked. You should use these commands to finish it

  • Connect the wifi

  • Test the wifi, you can use ping, for example

  • Update the system clock, you can use commands as follow

  • Mount the paritiion, this step maybe diffcult, because you can need the graphical guide. I find a video, you can refer to this.

Use fcitx on your archlinux

For me, the main editor is emacs, but in emacs, you use the Ctrl+\ to use chinese-pinyin and it is not enough smart to input the chinese. So I will tell you how to use fcitx5 in emacs and in your computer.

  • You should set the chinese is default
    • Alt F2, input setting, and then Regional Settings -> Language -> Add languages, choose the chinese and drag it to the first(before the en_US). Click the Apply
    • Then in the Formats choose the zh_CN(You can’t set language to mn_CN).
  • You should install the fcitx5 use command as follow
  • Then, set environment, use sudo emacs /etc/environment and input command as follow in emacs, use C-x C-s to save and the use C-x C-c to quit.
  • Then, Alt F2, input setting, and then Regional Settings -> Input methods, click on run fcitx, pinyin is the default options.

Now, you can use the fcitx in your emacs, before this, you should reboot to make the default environment is chinese.

Add shortcut for AppImage3

For ArchLinux, if you use pacman to install some softwares, it will install the AppImage file in the directory /opt/appimages/. So, before this, you can move your AppImage file to that directory with mv. In Linux, most of shortcuts are placed in /usr/share/applications/. Then if you don’t know the format of the *.desktop file, you can open a ‘*.AppImage’ file and copy, modify it.
