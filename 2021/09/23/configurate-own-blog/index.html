Posted on 2021-09-23 Edited on 2021-10-11 In Linux
Symbols count in article: 5.9k Reading time ≈ 5 mins.

Preface

First, I deploy my personal blog in Archlinux, so , if you use other systems, you should refer to other blog.
You should install nodejs, npm, cnpm, hexo and some related packages.
You don’t need to use sudo to install, or you will need root to write the .md.

Install

Step One

You should get a Github account. If you don’t know how to register it, you can use search engine to learn how to register.

Step Two

You should install git use command as following

<E-mail> is you github E-mail.

Step Three

You should generate and add ssh on you github.
First, you should need ssh-keygen -t rsa -C <Email>, then you can will get a invisible directory. In Linux, it on the main directory called .ssh/, in this directory, it has a file called id_rsa.pub.

  • Copy its contents.
  • Open your github setting, find the SSH and GPG keys, click the New SSH key to add new SSH.
  • You should check if you add successfully.
  • You use command ssh -T git@github.com to se the results. If your results just like following, you add successfully.

Step Four

  • You should install nodejs and npm
  • After, you should use node -v and npm -v to check their version,
  • If there are some warning, you should try it again, if they also has other issue, you can use the google engine.
  • You should use npm to install cnpm
  • But it will be slow, so you can use the repository from taobao, then use to install cnpm, also use cnpm -v to check if it is installed.
  • You should use cnpm to install hexo
    You can use to install hexo globally, also use hexo -v to check if hexo is installed.
  • Make a new directory to save your blogs
    For me, mkdir Blogs && cd Blogs to make a new directory and use it, then use command hexo init <username>.github.io to initialize a new directory.<username> is the name of your github.(Not neccessary, but recommend). I will introduce how to register a github account and add ssh to connent. Then, use c d <username>.github.io get into the directory.
  • See the blog
    hexo clean && hexo g && hexo s and you can open localhost:4000 to see the blog, you should use Ctrl+c to stop, if you can open it, use hexo s -p 5000 open the web on the port 5000.
  • Write a new blog
    Use hexo n "first" to get a new blog. Then use hexo clean && hexo g && hexo s to flush the blog.
